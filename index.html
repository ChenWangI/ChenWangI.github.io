前言

许多爱分享记录的朋友，通常会经历过这样的三个阶段。

第一个阶段，刚接触文章分享记录，觉得很新鲜，试着选择一个免费空间来写，常见的如QQ空间、微博、人人等等。

第二个阶段，发现免费空间限制的太多，接着就自己购买域名和空间，最后终于搭建好了独立的网页站，但是每年都交域名空间费用。

第三个阶段，觉得独立网页站的管理太麻烦，最好在保留控制权的前提下，可以让别人来接管，而我们自己只负责编写文章分享记录就好啦。

所以大多数分享记录的朋友，都停留在第一及第二阶段，因为第三阶段不太容易到达：你很难找到俯首听命、愿意为你管理服务器的人。

而就在近年来，情况出现了变化，许多朋友也在捣鼓，是否可以有这样一个，既是完全免费的，又无限流量的，能够保留控制权的，世界各地都可访问的网页站?

因此，这样的一个名为 Github 出现在了人们眼前，并且很快它所提供的，完全免费的，无限流量的，能够保留控制权的，世界各地都可访问的给力服务震惊了世界。
